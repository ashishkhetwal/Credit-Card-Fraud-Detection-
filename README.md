# Credit Card Fraud Detection System

A Machine Learning project that classifies credit card transactions as legitimate or fraudulent using Logistic Regression.

This project helped me understand how to work with highly imbalanced datasets and explore different techniques for handling class imbalance in classification problems.

## Concepts Used

* Data Preprocessing
* Feature Scaling
* Logistic Regression
* Imbalanced Dataset Handling
* Random Undersampling
* SMOTE (Synthetic Minority Oversampling Technique)
* Train-Test Split
* Classification Model Evaluation

## Results

* Built a Logistic Regression model for fraud detection.
* Explored both Undersampling and SMOTE techniques for handling class imbalance.
* Applied data scaling before training the model.
* Evaluated model performance on unseen test data.

## What I Learned

Through this project, I learned the importance of handling imbalanced datasets, the differences between Undersampling and SMOTE, and how oversampling techniques can help improve the training process. I also gained a better understanding of classification models and the importance of proper preprocessing and evaluation practices.
